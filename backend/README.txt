Backend folder

- server.js : backend entry wrapper
- api/chat.js : chatbot API route
- api/transcribe.js : voice transcription API route
- functions/chat.js : Netlify function copy for chat
- functions/transcribe.js : Netlify function copy for transcription
- .env.example : backend environment variable sample

Use this folder for server logic, API routes, deployment functions, and environment setup.
